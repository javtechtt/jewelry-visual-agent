# References

Drop visual references here that define the art direction — boutique
photography, lighting moodboards, product styling, competitor inspiration,
brand guidelines, etc.

These files are **not loaded by the app**. They are a design source-of-truth for
contributors. See `docs/ART_DIRECTION.md` for the target aesthetic.

Suggested structure:

```
references/
  moodboard/        # overall atmosphere, lighting, palette
  products/         # how each category should be styled/shot
  motion/           # transition + interaction references (gifs/clips)
  typography/       # type specimens
```
