# Textures

Optional textures for the environment and products: marble/floor maps, subtle
gradients, label artwork, normal maps, etc.

Reference them from a material via their public path, e.g. `/textures/marble.jpg`
loaded with drei's `useTexture`.

The starter ships with a fully **procedural** environment (no external HDRI), so
textures are optional — add them only when you want richer surfaces.
